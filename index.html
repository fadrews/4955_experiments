Token Legend (definitions)

COG
Core cognitive process the task targets (e.g., attention, working memory, valuation).
CTRL
Type of control or interference (e.g., inhibition, interference suppression).
MEM
Memory processes involved (span, updating, short-term storage).
ATTN
Attentional demands (sustained, selective, divided).
DEC
Decision-making mechanism (risk-taking, binary choice, detection).
MOT
Motor / perceptual response attributes (keypress, pointing, aiming).
DYN
Trial dynamics & structure (staircase, SOA manipulation, learning across trials).
MEAS
Typical dependent measures (RT, accuracy, SSRT, lapses).
